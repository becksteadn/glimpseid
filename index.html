About

I used urlscan.io a lot at my internship. Afterwards I built HTTP Info, a Docker container to screenshot a website and record HTTP requests and responses that occur when visiting the site.

This is an extension of that project. Glimpse is a cheap, scalable API that utilizes multiple AWS services such as Lambda, S3, and API Gateway.
